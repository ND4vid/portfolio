< Processor Inventory API />

Overview

RESTful API built with Laravel for managing processor inventory. The application provides complete CRUD functionality, input validation, and relational database management using Eloquent ORM, serving as a scalable foundation for inventory-based systems.

Laravel 12 PHP REST API MySQL Eloquent ORM CRUD Database Relationships

Key Features

  • Manage processor records through a RESTful API
  • Perform complete CRUD operations
  • Associate processors with CPU brands using Eloquent relationships
  • Validate incoming request data before database operations
  • Return structured JSON responses with appropriate HTTP status codes
  • Seed the database with sample data for testing and development
  • Follow Laravel's MVC architecture and RESTful conventions

API Endpoints

Method Endpoint Description
GET /processors Retrieve all processors with their associated CPU brands.
GET /processors/{id} Retrieve a specific processor (with its associated CPU brand).
POST /processors Create a new processor.
PUT /processors/{id} Update an existing processor.
DELETE /processors/{id} Delete a processor.
